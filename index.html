New HTML Element Details

I chose to use the <details> element which creates a disclosure widget that can be opened or closed by the user. This is my first time using it, and I learned that it's a great way to hide content that isn't immediately necessary but can be accessed with a click.

It works well for FAQs, additional information, or any content that might clutter the page if always visible.

New CSS Property Exploration

I chose to explore the CSS property backdrop-filter, which allows you to apply graphical effects like blurring to the area behind an element. It's useful for creating frosted glass effects and maintaining readability when text overlaps with busy backgrounds.
